## 📱 功能说明

### 底部导航栏
现在小程序底部有 3 个标签页：
1. **日历** - 查看日历、预约情况
2. **活动** - 浏览和报名活动
3. **我的** - 个人信息、订单、会员

### 登录流程
1. 点击"我的"页面
2. 点击"登录/注册"
3. 输入手机号和密码
4. 登录后可享受会员优惠

## ⚠️ 关于图标

当前使用 emoji 作为临时图标。如需恢复原始图标：

### 方案 1: 使用原始图标文件
如果有原始图标文件，放入 `images/` 目录：
- fish.png / fish-active.png
- calendar.png / calendar-active.png  
- user.png / user-active.png

### 方案 2: 修改 app.json 使用图标
```json
"tabBar": {
  "list": [
    {
      "pagePath": "pages/index/index",
      "text": "日历",
      "iconPath": "images/fish.png",
      "selectedIconPath": "images/fish-active.png"
    }
  ]
}
```
