# 解决登录"网络错误"

## 问题分析
环境 ID 已确认正确：`cloud1-3g3ropz587d6e804`

最可能的原因是：**数据库集合未创建**

## 🔧 解决方案（2 选 1）

### 方案 A：手动创建集合（推荐，简单）

1. **打开云开发控制台**
   - 微信开发者工具 → 点击"云开发"按钮

2. **创建数据库集合**
   - 进入"数据库"标签
   - 点击"+"创建以下 4 个集合：
     - `users`
     - `bookings`
     - `trips`
     - `activityTemplates`

3. **设置集合权限**
   - 点击每个集合 → "权限设置"
   - 选择"所有用户可读，仅创建者可读写"
   - 或自定义权限：
   ```json
   {
     "read": true,
     "write": true
   }
   ```

4. **重新测试登录**

---

### 方案 B：使用云函数自动创建

1. **上传云函数**
   - 在微信开发者工具中
   - 右键 `cloudfunctions/initDatabase` 目录
   - 选择"上传并部署：云端安装依赖"

2. **调用云函数**
   - 在小程序控制台执行：
   ```javascript
   wx.cloud.callFunction({
     name: 'initDatabase',
     success: console.log,
     fail: console.error
   })
   ```

3. **查看结果**
   - 显示哪些集合已创建或已存在

---

## ✅ 验证步骤

在微信开发者工具控制台执行：

```javascript
// 1. 检查云开发
wx.cloud  // 应该显示 cloud 对象

// 2. 检查数据库
wx.cloud.database()  // 应该显示 database 对象

// 3. 测试查询 users
wx.cloud.database().collection('users').limit(1).get()
  .then(res => console.log('成功:', res))
  .catch(err => console.error('失败:', err))
```

如果第 3 步成功，说明云开发配置正确，可以正常登录了。
