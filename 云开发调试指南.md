# 云开发调试指南

## 问题：登录提示"网络错误"

### 可能原因

1. **云开发环境未开通**
2. **环境 ID 错误**
3. **数据库集合权限未配置**
4. **AppID 不匹配**

## 检查步骤

### 1. 确认云开发环境

当前配置的环境 ID：
```
cloud1-3g3ropz587d6e804
```

在微信开发者工具中：
1. 点击工具栏"云开发"按钮
2. 查看环境 ID 是否匹配
3. 如果不匹配，修改 `app.js` 中的 `cloudEnv`

### 2. 创建数据库集合

在云开发控制台 -> 数据库，创建以下集合：

| 集合名 | 权限 |
|--------|------|
| users | 所有用户可读，仅创建者可读写 |
| bookings | 所有用户可读，仅创建者可读写 |
| trips | 所有用户可读，仅创建者可读写 |
| activityTemplates | 所有用户可读，仅创建者可读写 |

### 3. 配置数据库权限

**users 集合权限建议**：
```json
{
  "read": "auth.openid == doc.openid",
  "write": "auth.openid == doc.openid"
}
```

### 4. 检查 AppID

当前配置：
```
wx504a5b83dba5ce0b
```

确保：
- 微信开发者工具中登录的账号有权限
- 云开发环境属于该 AppID

## 快速测试

在微信开发者工具控制台中执行：

```javascript
// 测试云开发初始化
wx.cloud.callFunction({
  name: 'test',
  success: console.log,
  fail: console.error
})

// 测试数据库连接
wx.cloud.database().collection('users').limit(1).get().then(console.log).catch(console.error)
```

## 临时解决方案（本地存储）

如果云开发有问题，可以先用本地存储测试登录功能。

## 修改环境 ID

如果环境 ID 不对，修改以下文件：

**app.js**:
```javascript
globalData: {
  cloudEnv: '你的环境 ID'
}
```
